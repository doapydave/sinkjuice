A premium add-on pack for Store Locator Plus that adds an enhanced map interface.

== Description ==

Get more control of the map interface on Store Locator Plus with this add-on.

= Features =

* Turn map on/off with a shortcode.
* Map show/hide setting.

== Installation ==

= Requirements =

* Store Locator Plus: 4.1+
* WordPress: 3.3.2+
* PHP: 5.1+

= Install After SLP =

1. Go fetch and install Store Locator Plus version 3.8.2 or higher.
2. Purchase this plugin from CSA to get the latest .zip file.
3. Go to plugins/add new.
4. Select upload.
5. Upload the slp-enhanced-map.zip file.
