A premium add-on pack for Store Locator Plus that adds business icons to the icon pickers.

== Description ==

A free add-on pack for the [Store Locator Plus](http://www.charlestonsw.com/) location mapping system plugin.
Icons will appear in the default home or end marker selection as well as the extended
per-location map markers offered by [Enhanced Map](http://www.charlestonsw.com/product/slp4-enhanced-map/) or the categorical markers offered
via [Tagalong](http://www.charlestonsw.com/product/slp4-tagalong/).

* More icons.  That's all.

== Installation ==

= Requirements =

* Store Locator Plus: 4.0+
* Wordpress: 3.3.2+
* PHP: 5.1+

= Install After SLP =

1. Go fetch and install Store Locator Plus version 4.0 or higher.
2. Purchase this plugin from CSA to get the latest .zip file.
3. Go to plugins/add new.
4. Select upload.
5. Upload the zip file.
