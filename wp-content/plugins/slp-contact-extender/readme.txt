A premium add-on pack for Store Locator Plus that adds contact fields to the location data set.

== Description ==

Add contact data fields to the locations in Store Locator Plus.

= Features =

Adds the following fields to location data:

* Identifier
* Contact
* First Name
* Last Name
* Title
* Department
* Office Phone
* Mobile Phone
* Contact Fax
* Contact Email
* Office Hours
* Contact Address
* Notes

== Installation ==

= Requirements =

* Store Locator Plus: 4.1+
* Wordpress: 3.3.2+
* PHP: 5.2.4+

= Install After SLP =

1. Go fetch and install Store Locator Plus version 4.1 or higher.
2. Purchase this plugin from CSA to get the latest .zip file.
3. Go to plugins/add new.
4. Select upload.
5. Upload the zip file.
