A premium add-on pack for Store Locator Plus that adds social media functionality.

== Description ==

Allows management of social media settings.
Adds social media account information to individual locations.
Show configured social media icons with the search results.

= Features =

Adds an additional field to location data for each social media supported.

== Frequently Asked Questions ==

= How does the add-on work? =

The add-on adds a new page to the set of SLP configuration pages.
The 'Social Media Extender' page presents the Admin with a list of configured social settings.
The Admin can add and edit social settings, e.g. to define the icon and base url.
For each social media available, the location information contains a field to store the social media account.
For each social media account, the search results shows the associated icon which can be clicked to address the social media account.
