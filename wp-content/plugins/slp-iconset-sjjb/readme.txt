== Installation ==

= Requirements =

* Store Locator Plus: 3.8.7+
* Wordpress: 3.3.2+
* PHP: 5.1+

= Install After SLP =

1. Go fetch and install Store Locator Plus version 3.8.7 or higher.
2. Purchase this plugin from CSA to get the latest .zip file.
3. Go to plugins/add new.
4. Select upload.
5. Upload the zip file.
